# 2048 Premium Edition

A modern and responsive 2048 game built with pure HTML, CSS, and JavaScript.

## Features

- Classic 2048 gameplay
- Mobile-first responsive design
- Desktop and mobile layouts
- Smooth tile animations
- Win screen with "Keep Going" mode
- Undo last move
- Local save and resume
- Statistics tracking
- Achievement system
- Best score tracking
- Highest tile tracking
- Fastest win tracking
- Beautiful glassmorphism UI
- Confetti celebration on win
- No frameworks required

## How to Play

1. Use Arrow Keys (Desktop) or Swipe (Mobile).
2. Merge tiles with the same number.
3. Reach the 2048 tile to win.
4. Continue playing for a higher score after winning.
5. Try to beat your Best Score and unlock all achievements.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage

## Controls

| Action | Control |
|----------|----------|
| Move Left | ← Arrow Key |
| Move Right | → Arrow Key |
| Move Up | ↑ Arrow Key |
| Move Down | ↓ Arrow Key |
| Undo | Undo Button |
| New Game | New Game Button |

## Achievements

- First Merge
- Reach 128
- Reach 512
- Reach 1024
- Reach 2048
- Score 10,000
- Play 10 Games
- Play 50 Games

## Data Storage

Game progress, statistics, achievements, and settings are stored locally in your browser using Local Storage.
